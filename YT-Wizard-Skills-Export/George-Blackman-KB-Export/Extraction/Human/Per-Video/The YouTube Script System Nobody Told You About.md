# The YouTube Script System Nobody Told You About

## What This Video Mainly Teaches
A script becomes easier when you build it as a visible system of segments, write the body before the hook, and use a repeatable method for tension.

## Strongest Takeaways
- Use a grid or table to keep the script movable and visible.
- Write payoffs first, then setups, then tension, then hook.
- Build hooks from target, transformation, and stakes after the body is clearer.
- Treat end-screen CTAs as a structured part of the script.

## Useful Frameworks
- Setup -> tension -> payoff
- Payoffs first
- Target -> transformation -> stakes
- Link -> curiosity -> promise

## Useful Examples
- The video itself demonstrates the system in order.

## Anti-Patterns / Warnings
- Writing linearly from the hook down
- Treating tension as filler instead of guided development

## Best Phrases Worth Keeping
> "payoffs first"
