## What This Video Mainly Teaches
Curiosity helps retention only when each question is resolved before the next major question opens.

## Strongest Takeaways
- More curiosity gaps are not automatically better.
- Nested unresolved curiosity gaps confuse the viewer.
- Each setup should lead to one payoff before the next setup begins.

## Useful Frameworks
- Curiosity gap vs curiosity cavern
- Setup -> tension -> payoff

## Useful Examples
- The gardening/parsnip example shows how stacked curiosity gaps become confusing.

## Anti-Patterns / Warnings
- Curiosity gaps inside curiosity gaps
- Letting the viewer lose the main thing they are supposed to focus on
