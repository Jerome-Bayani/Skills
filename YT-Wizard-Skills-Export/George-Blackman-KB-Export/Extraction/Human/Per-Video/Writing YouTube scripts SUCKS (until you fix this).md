# Writing YouTube scripts SUCKS (until you fix this)

## What This Video Mainly Teaches
Scriptwriting becomes much easier when you handle audience, framing, and payoff structure before trying to write the full draft.

## Strongest Takeaways
- Build a real audience avatar and anti-avatar.
- Frame the idea for your actual audience before drafting.
- Plot payoffs before trying to write the full script.

## Useful Frameworks
- Audience avatar
- Framing checklist
- Payoff-first skeleton

## Useful Examples
- The paragliding/skiing example shows how a creator-centered concept can misalign with the actual audience.

## Anti-Patterns / Warnings
- Starting with a cool idea before checking audience fit
- Using vague subtopics instead of clear payoffs

## Best Phrases Worth Keeping
> "actually writing the script is the fourth of those five steps"
