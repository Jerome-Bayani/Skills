# The Clever Way Smart YouTubers Make Viewers BINGE

## What This Video Mainly Teaches
A strong end-screen CTA can create a binge loop if it feels like an offer tied to the current video rather than a generic ask.

## Strongest Takeaways
- Do not use the very end to ask for likes and subscriptions.
- Recommend a next video that goes deeper on a topic just introduced.
- Use link, curiosity, and promise in sequence.

## Useful Frameworks
- Link -> curiosity -> promise
- Offer, not ask

## Useful Examples
- The newsletter recommendation example shows how to point to a deeper adjacent payoff.

## Anti-Patterns / Warnings
- Generic "watch this next" CTAs
- Treating the end screen as a favor request

## Best Phrases Worth Keeping
> "asking them to watch your next video is an offer"
