# Skill Opportunities

## Highest-Leverage Skill Ideas

- Channel best-practices extractor for scriptwriting channels
- Video script analyzer for strong videos in any topic
- Framework detector for identifying named and inferred script structures
- Writer skill builder that turns extracted frameworks into a reusable writing workflow
- AI collaboration optimizer for segmented script drafting and refinement

## Why These Skill Ideas Matter

- The channel repeatedly turns performance observations into rules
- The material is already structured around frameworks and repeatable decisions
- The advice is practical enough to become SOP-like behavior inside a skill
- AI usage is framed as process support, which makes it easier to operationalize
