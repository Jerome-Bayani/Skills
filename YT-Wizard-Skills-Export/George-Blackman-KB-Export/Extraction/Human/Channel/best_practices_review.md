# Best Practices Review

## Core Best Practices

- Build scripts around setup, tension, and payoff
- Write payoffs first so the structure is visible early
- Write hooks, setups, and payoffs word for word, then let the middle flex
- Add stakes to hooks and setups, not just the video premise
- Use retention analysis to create reusable writing rules
- Make the structure visually obvious to the viewer during the video
- Break AI-assisted writing into smaller tasks instead of one giant prompt
- Keep the final judgment pass human

## Important Best Practices

- Use grid or table formatting so segments are easy to move and audit
- Match scripting style to delivery style, teleprompter use, and personality
- Separate research from writing to avoid context switching
- Use a small set of proven hook patterns instead of reinventing the wheel
- Build end-screen CTAs as offers, not asks
- Use external eyes or a community review before filming
- Turn repeated findings into frameworks, not just tips

## Supporting Best Practices

- Try multiple formats to speed up learning
- Use "blob scripts" or placeholders to protect momentum
- Use visual anchors to re-orient viewers during longer videos
- Use exact phrases selectively when they carry worldview or memorability

## Repeated Patterns Worth Special Attention

- Structure beats inspiration
- Clarity beats cleverness
- Retention problems often begin earlier than the visible dip
- Better scripts come from stronger diagnosis, not just more effort
