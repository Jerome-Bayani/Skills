# Process Workflows

## Script Construction Workflow

1. Clarify audience and framing
2. Decide the segment payoffs
3. Write the setups that make those payoffs matter
4. Fill the tension phase without giving away the answer too early
5. Write the hook after the body is clearer
6. Add the end-screen CTA
7. Review for retention risks and confusion

## Retention Diagnosis Workflow

1. Look at the visible dip
2. Check whether the hook set a wrong format expectation
3. Check whether the previous setup was weak or muddy
4. Check whether the previous payoff under-delivered
5. Convert the finding into a reusable writing rule

## AI-Assisted Writing Workflow

1. Brainstorm with AI instead of asking for a full script immediately
2. Settle the angle and the structure collaboratively
3. Draft one script task at a time
4. Rewrite segment one manually
5. Feed those changes back into the AI
6. Repeat segment by segment
7. Perform the final 10 percent manually
8. Generate a learning report from the differences
9. Update the style guide and training document

## Speed Workflow

1. Choose bullet-point or word-for-word scripting based on delivery needs
2. Separate research and writing
3. Use templates for hooks and segment structures
4. Use placeholder language to preserve momentum
5. Timebox the writing session
6. Limit revisions
