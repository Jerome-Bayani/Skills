# Frameworks

## Named Frameworks

- name: setup -> tension -> payoff
  description: Each segment should promise, develop, and then resolve one idea in sequence.
  recurring: true
  importance: core

- name: target -> transformation -> stakes
  description: Hook framework for educational videos.
  recurring: true
  importance: core

- name: character -> concept -> stakes
  description: Hook framework for entertainment-style videos.
  recurring: false
  importance: important

- name: scriptwriting dial
  description: Adjust how tightly scripted the video is based on delivery needs.
  recurring: false
  importance: important

- name: blob script
  description: Protect momentum by using placeholders instead of stopping for perfect wording.
  recurring: false
  importance: important

- name: link -> curiosity -> promise
  description: End-screen CTA structure that turns a next-video recommendation into an offer.
  recurring: true
  importance: important

## Inferred Frameworks

- name: retention graph review loop
  description: Use retention data to detect repeated problems, then convert those patterns into rules for future scripts.
  recurring: true
  importance: core
  confidence: high

- name: segment-first drafting
  description: Build the script as a set of segments with visible payoffs before worrying about perfect prose.
  recurring: true
  importance: core
  confidence: high

- name: AI collaboration loop
  description: Brainstorm with AI, draft segment by segment, edit manually, then feed the edits back into the system.
  recurring: true
  importance: core
  confidence: high

- name: visual orientation loop
  description: Use checklists, maps, blueprints, or recurring cues to repeatedly tell the viewer where they are in the video.
  recurring: true
  importance: important
  confidence: high
