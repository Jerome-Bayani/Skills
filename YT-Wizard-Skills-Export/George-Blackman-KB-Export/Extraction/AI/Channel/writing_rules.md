# Writing Rules

- rule: Never reveal the payoff early and then keep talking around it.
  importance: core
  recurring: true

- rule: Every setup should answer "why should I care?" for the next segment.
  importance: core
  recurring: true

- rule: Do not open curiosity gaps inside other unresolved curiosity gaps.
  importance: core
  recurring: true

- rule: Keep hooks to critical information only. Move beneficial information later.
  importance: core
  recurring: true

- rule: Give the viewer something meaningful to look forward to at least every couple of minutes.
  importance: core
  recurring: true

- rule: If a segment is confusing, inspect the setup before it and the payoff before that.
  importance: important
  recurring: true

- rule: If you can templatize the structure, spend your brainpower on insights instead of reinvention.
  importance: important
  recurring: true

- rule: Use exact wording selectively. Use systems everywhere.
  importance: supporting
  recurring: false

- rule: Use two focused revisions, then ship.
  importance: important
  recurring: false
